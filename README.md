# Телеграм бот для просмотра и загрузки файлов   

## команды:
- `/help` -список команд

- `/ls` - список файлов текущей директории

- `/pwd` - отображение  текущей директории

- `/cd` - перемещение по директориям, пример: `/cd папка`

- `/download` - отправка файлов, пример: `/download файл` 

- `/upload` - загрузка файла на сервер с ботам, для загрузки необходимо отправить боту документ фото или видео далее нужно ответить командой `/upload` на это сообщение

- `/conloe` - исполнение команд в консоли

- `/test` - отображает информацию о сервере такую как нагрузка/запылённость  CPU, RAM, snap file, пинг   

## настройка:

файл config.json

```json
{
    "tokin":" ",
    "users":[5194033781],
    "dir":null
}
```

`tokin` -токен Телеграм бота

`users` -список с id пользователей которые могут использовать бота не считая команды /help она общедоступна

`dir` -рабочая директория она при старте. если указать null то будет использована директория запуска приложения
